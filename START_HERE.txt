📋 STEP-BY-STEP INSTRUCTIONS:

1️⃣  START BACKEND (Terminal 1)
   
   cd backend
   venv\Scripts\activate
   python main.py
   
   ✅ Backend running at: http://localhost:8000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2️⃣  START FRONTEND (Terminal 2 - NEW WINDOW)
   
   cd frontend
   npm install
   npm run dev
   
   ✅ Frontend running at: http://localhost:3000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3️⃣  OPEN IN BROWSER
   
   Visit: http://localhost:3000
   
   You should see the Dormo landing page! 🎉

🐛 TROUBLESHOOTING:

Problem: "Failed to resolve import"
Solution: Make sure you're in the "frontend" folder, not "my-react-app"

Problem: Backend won't start
Solution: Check if Python 3.9+ is installed and virtual environment is activated

Problem: Frontend won't start
Solution: Run "npm install" first in the frontend folder
